Giveaway Commands

Commands to create, manage, and end giveaways in your server.

Default Prefix: . (The main group command is giveaway or g)

{/* Note: The python command uses @giveaway.command() so the callable command is `.g start` or `.giveaway start` */}

.g start / .giveaway start

Syntax: .g start [@RoleMention]

Aliases: .g create, .giveaway create

Starts a new giveaway in the current channel.

Required Permissions:

  • Bot: Send Messages, Embed Links, Add Reactions, Read Message History, Manage Messages (to delete command)
  • User: Manage Guild

Parameters:

  • (Required): How long the giveaway should last (e.g., 10s, 5m, 1h, 3d, 1w, 1h30m). Minimum 10s.
  • (Required): How many winners to draw (must be 1 or more).
  • (Required): What the prize is (max 150 chars).
  • [@RoleMention] (Optional): Add a role mention at the end of the prize name to require that role for entry.

Examples:

.g start 1d 1 Nitro Classic
.g start 30m 3 "Custom Role"
.g start 1w 1 Awesome Game Key @Level 10+

.g list / .giveaway list

Syntax: .g list

Aliases: None

Lists the currently active giveaways in the server.

Required Permissions:

  • Bot: Send Messages, Embed Links
  • User: None

Examples:

.g list

.g end / .giveaway end

Syntax: .g end

Aliases: None

Immediately ends a specific giveaway and draws winners.

Required Permissions:

  • User: Manage Guild

Parameters:

  • (Required): The ID of the giveaway message to end.

Examples:

.g end 112233445566778899

.g reroll / .giveaway reroll

Syntax: .g reroll [number_to_reroll]

Aliases: None

Picks new winner(s) for a giveaway that has already ended.

Required Permissions:

  • User: Manage Guild

Parameters:

  • (Required): The ID of the *ended* giveaway message.
  • [number_to_reroll] (Optional): How many new winners to draw (Default: 1).

Examples:

.g reroll 112233445566778899
.g reroll 998877665544332211 2
