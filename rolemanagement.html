Role Management Commands

Commands for managing roles, including setting up reaction roles.

Default Prefix: .

.reactionrole / .rr

Syntax: .reactionrole [subcommand] [options...]

Sets up or manages messages where users can react with an emoji to gain or lose a specific role.

Use subcommands like .rr create, .rr add, .rr remove, .rr list. See details below.
