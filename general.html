General Commands

Basic utility and informational commands.

Default Prefix: .

.help

Syntax: .help [category/command]

Displays a list of command categories or detailed information about a specific command/category.

Parameters:

  • [category/command] (Optional): The name of a command category (e.g., Moderation) or a specific command (e.g., kick) to get more details.

Examples:

.help
.help Moderation
.help kick

.ping

Syntax: .ping

Checks the bot's current response time (latency) to Discord.

Examples:

.ping

The bot will respond with something like "Pong! `[XX]`ms".

.hello

Syntax: .hello

Aliases: .chào, .lô

A simple command to make the bot say hello back to you.

Examples:

.hello

The bot will respond with "Hello @YourUsername!".
