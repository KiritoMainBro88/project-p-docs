Giveaway Commands

Commands to create, manage, and end giveaways in your server.

Default Prefix: .

.gstart

Syntax: .gstart [--channel #channel] [--reqrole @Role]

Starts a new giveaway in the current or a specified channel.

Required Permissions:

  • Bot: Send Messages, Embed Links, Add Reactions, Read Message History
  • User: [Permissions needed, e.g., Manage Server, or a specific Giveaway Host Role]

Parameters:

  • (Required): How long the giveaway should last (e.g., 10m, 1h, 3d, 1w).
  • (Required): How many winners to draw (e.g., 1, 3w).
  • (Required): What the prize is (e.g., Nitro Classic, Custom Role).
  • [--channel #channel] (Optional): The channel where the giveaway message should be posted. Defaults to the current channel.
  • [--reqrole @Role] (Optional): A role users must have to enter the giveaway.

Examples:

.gstart 1d 1w Steam Key for Cool Game
.gstart 30m 3w "Special Server Role (30 days)" --channel #giveaways
.gstart 1w 1w Lifetime VIP Access --reqrole @Level10+

.glist

Syntax: .glist [--channel #channel]

Lists the currently active giveaways in the server or a specific channel.

Required Permissions:

  • Bot: Send Messages, Embed Links
  • User: [Usually none, or basic access]

Parameters:

  • [--channel #channel] (Optional): Only list giveaways running in this specific channel.

Examples:

.glist
.glist --channel #giveaways
