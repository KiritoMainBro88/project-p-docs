Stream Monitor Commands

Configure notifications for when YouTube or TikTok creators go live.

Default Prefix: .

.notify add

Syntax: .notify add [#channel]

Adds a creator to the notification list for a channel. The bot will post a message when they go live.

Required Permissions:

  • User: Manage Guild

Parameters:

  • (Required): The platform ('youtube' or 'tiktok').
  • (Required): The creator's identifier. Examples:
    • YouTube: Channel name (`MrBeast`), handle (`@MrBeast`), channel ID (`UC...`), full channel URL.
    • TikTok: Username (`charlidamelio`), full profile URL.
  • [#channel] (Optional): The text channel where notifications should be sent. Defaults to the current channel.

Examples:

.notify add youtube MrBeast #streams
.notify add tiktok charlidamelio
.notify add youtube https://www.youtube.com/channel/UCX6OQ3DkcsbYNE6H8uQQuVA #general

Note: TikTok detection can be less reliable than YouTube.

.notify remove

Syntax: .notify remove [#channel]

Aliases: .notify delete

Removes a creator from the notification list for a channel.

Required Permissions:

  • User: Manage Guild

Parameters:

  • (Required): The platform.
  • (Required): The identifier used when adding (or check .notify list).
  • [#channel] (Optional): The channel the notification was configured for. Defaults to the current channel.

Examples:

.notify remove youtube MrBeast #streams

.notify list

Syntax: .notify list [#channel]

Lists all creators being monitored for notifications in a specific channel (or the current one).

Required Permissions:

  • User: None (or Manage Guild, depending on desired restriction)

Parameters:

  • [#channel] (Optional): The channel to list notifications for. Defaults to current channel.

Examples:

.notify list #live-streams

.notify check

Syntax: .notify check [#channel]

Aliases: .notify status

Manually checks the current live status of a creator already added to the notification list.

Required Permissions:

  • User: Manage Guild

Parameters:

  • (Required): The platform.
  • (Required): The identifier used when adding.
  • [#channel] (Optional): The channel the notification was configured for. Defaults to current channel.

Examples:

.notify check youtube MrBeast
