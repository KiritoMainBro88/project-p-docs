Levels & Rank Commands

Commands related to the server's XP and leveling system. XP is typically gained by sending messages (with a cooldown).

Default Prefix: .

{/* Ensure this ID matches search-index.json */}

.rank

Syntax: .rank [member]

Aliases: .level

Shows your current XP, level, and server rank, or that of another member. Displays a graphical rank card if image generation is enabled and working.

Required Permissions:

  • User: None

Parameters:

  • [member] (Optional): The user whose rank to view (@Mention, ID, Username#Tag). Defaults to yourself.

Examples:

.rank
.rank @User123
.level KiritoMainBro

Note: The leveling system can be disabled by server administrators.

.leaderboard

Syntax: .leaderboard [page]

Aliases: .lb, .top

Displays the server's XP leaderboard, showing top-ranked members.

Required Permissions:

  • User: None

Parameters:

  • [page] (Optional): The page number of the leaderboard to view (Default: 1).

Examples:

.leaderboard
.lb 2
.top
