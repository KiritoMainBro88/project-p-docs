Getting Started with Project P

1. Inviting the Bot

Adding Project P to your server is simple:

  1. Click the Invite Link: Use this link to start the process:
    Invite Project P to Your Server
  2. Select Your Server: Choose the Discord server you want to add the bot to from the dropdown menu. You must have "Manage Server" permissions on that server.
  3. Authorize Permissions: Review the permissions Project P requests. These are necessary for its commands to function correctly (e.g., `Kick Members` for `.kick`, `Manage Roles` for `.reactionrole`, `Manage Messages` for `.purge`). Click "Authorize".
  4. Confirmation: You might need to complete a CAPTCHA. Once done, the bot should appear in your server's member list!

2. Understanding the Prefix

Commands for Project P start with a prefix. The default prefix is a dot: .

To use a command, type the prefix followed immediately by the command name. For example: .help, .ping, .kick @User.
