Embed Builder Commands

Create, edit, and send custom rich embed messages.

Default Prefix: .

{/* Matched to potential search index key */}

.embed create

Syntax: .embed create

Creates a new, named embed definition in the database.

Required Permissions:

  • User: Manage Messages

Parameters:

  • (Required): A unique name for the embed (no spaces, max 50 chars).

Examples:

.embed create rules-embed
{/* Matched to potential search index key */}

.embed edit

Syntax: .embed edit

Edits a specific part of a saved embed definition.

Required Permissions:

  • User: Manage Messages

Parameters:

  • (Required): The name of the embed to edit.
  • (Required): The part of the embed to change (see list below).
  • [value] (Depends on option): The new value. Omit to clear some options.

Available Options:

  • title - Sets the main title.
  • description - Sets the main text body (supports Markdown).
  • color <#hex or none> - Sets the side color (e.g., #FF0000 for red).
  • url - Makes the title a hyperlink.
  • author_name - Sets the text for the author line (top).
  • author_icon - Sets the small icon next to the author name.
  • author_url - Makes the author name a hyperlink.
  • footer_text - Sets the text for the footer line (bottom).
  • footer_icon - Sets the small icon next to the footer text.
  • thumbnail - Sets the small image in the top-right corner.
  • image - Sets the large image displayed below the content.
  • addfield | [| inline:true] - Adds a field. Use `|` to separate name and value. Add `| inline:true` at the end to make it inline (max 3 inline fields per row).
  • removefield - Removes a field by its number (starting from 1).
  • clearfields - Removes all fields from the embed.

Examples:

.embed edit rules-embed title Server Rules
.embed edit rules-embed color #ffcc00
.embed edit rules-embed description Please read and follow these rules.
.embed edit rules-embed addfield Rule 1 | Be respectful to everyone.
.embed edit rules-embed addfield Links | [Website](...) | inline:true
.embed edit rules-embed removefield 1
.embed edit rules-embed image none

.embed view

Syntax: .embed view

Shows a preview of the specified custom embed.

Required Permissions:

  • User: Manage Messages

Parameters:

  • (Required): The name of the embed to preview.

Examples:

.embed view rules-embed
{/* Matched to potential search index key */}

.embed send

Syntax: .embed send [#channel]

Sends the specified custom embed to a channel.

Required Permissions:

  • User: Manage Messages
  • Bot: Send Messages, Embed Links (in the target channel)

Parameters:

  • (Required): The name of the embed to send.
  • [#channel] (Optional): The text channel to send the embed to. Defaults to the current channel.

Examples:

.embed send rules-embed
.embed send welcome-embed #welcome

.embed delete

Syntax: .embed delete

Deletes a saved embed definition permanently.

Required Permissions:

  • User: Manage Messages

Parameters:

  • (Required): The name of the embed definition to delete.

Examples:
