Reputation Commands

Commands for giving and viewing user reputation points.

Default Prefix: .

.rep

Syntax: .rep

Aliases: .+rep

Gives one reputation point to another member. Subject to a cooldown (typically 24 hours, configurable by admin).

Required Permissions:

  • User: None

Parameters:

  • (Required): The user to give reputation to (@Mention, ID, Username#Tag). Cannot be yourself or a bot.

Examples:

.rep @HelpfulUser

Note: Reputation system can be disabled by server administrators.

.repleaderboard

Syntax: .repleaderboard [page]

Aliases: .reps, .repboard

Displays the server's reputation leaderboard.

Required Permissions:

  • User: None

Parameters:

  • [page] (Optional): The page number of the leaderboard to view (Default: 1).

Examples:

.repleaderboard
.reps 2
.repboard 1
